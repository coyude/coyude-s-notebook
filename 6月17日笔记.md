（1）int类型不遵守四舍五入。3.5=3；
（2）如果有一个数据是float或double型，其他数据类型先转化为double型，得到的结果最终也是double型；
（3）a++ 与 ++a简单分析
1.++a是先进行自增/自减运算，然后再进行其它运算。
2.a++后自增/后自减，即先进行其它运算，当其它运算都完成后，再进行自增/自减运算。
（4）优先级：

1.算术运算术的优先级要高关系运算符；关系运算符的优先级要高于逻辑运算符
2.逻辑非优先级最高。逻辑与要高于逻辑或。
3.如果在不确定，就加小括号。

（5）
*p和**p的区别
int *p ：一级指针，表示p所指向的地址里面存放的是一个int类型的值
int **p ：二级指针，表示p所指向的地址里面存放的是一个指向int类型的指针（即p指向的地址里面存放的是一个指向int的一级指针）
例如：
int i=10; //定义了一个整型变量
int *p=&i; //定义了一个指针指向这个变量
int **p1=&p; //定义了一个二级指针指向p指针
那么取出10的值方式为：
printf(“i=[%d]\n”,*p);
printf(“i=[%d]\n”,**p1);

（7）
switch用法



```c
int s=1，k=0;
switch (s)
{
    case 1: k += 1;
	case 2: k += 2; break;
}
```



输出的K值是3，此为switch特性，执行完case 1的语句后后，会继续执行下面的句子，直到break出现方能跳出循环。正确做法是case 1: k += 1;break;

字符串转成整数：sscanf(*str,”%d”,&x);
Sample:



```c
#include<stdio.h>
int main(void)
{
    int x = 0;
	const char* str = "12345";
	sscanf(str, "%d", &x);
	printf("%d", x);
	return 0;
}
```
