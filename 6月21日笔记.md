## 线性存储结构（又称线性表）【数据结构的一种】

线性表是由同一类型的数据元素构成的有序序列的线性结构，其分为**顺序存储结构**和**链式存储结构**两种。

目前笔者所学的**数组**属于**顺序存储结构**，**链表**属于**链式存储结构**。

两者最大的区别在于，顺序存储结构要求物理上**相邻**（即地址连续的存储单元），链式储存结构则不做要求。基于这个特点，**顺序表**(顺序存储结构的线性表)在插入/删除元素时需要通过移动数据元素来实现，影响效率。

### 链式表

前置条件：

```c
typedef struct LNode* PtrToLNode;
struct LNode{
    ElemenType Data;
    PtrToLNode Next;
};
typedef PtrLNode Position;/*位置是结点的地址*/
typedef PtrLNode List;
```



**(1)求表长**

```c
int length(List L)
{
    Position p;
    int cnt=0;
    p=L;
    while(p)
    {
        p=p->Next;
        cnt++;
    }
    return cnt;
}
```
