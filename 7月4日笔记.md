## C++11中raw string literal

raw string literal最大的特点就是：它不会对反斜杠'\'进行特殊的转义处理。

使用方法：R"(XXXXX)"

```c++
#include<iostream>
using namespace std;
int main()
{
	cout << R"(114514\n)";
	cout << "1919810";
	return 0;
}
```

输出：

```
114514\n1919810
```

可见对转义符\n不进行处理，保证了字符串的原汁原味()

## 最大子列和（在线处理）

直接上代码：

```c++
int Maxsubseqsum(int list[], int N)
{
	int thissum, maxsum;
	thissum = maxsum = 0;
	for (int i = 0; i < N; i++)
	{
		thissum += list[i];
		if (thissum > maxsum)
			maxsum = thissum;
		else if (thissum < 0)//当前子列和为负
			thissum = 0;//不可能使后面的部分增大，抛弃这一段
	}
	return maxsum;
}
```

这一方法精髓在于当thissum<0时，抛弃这一段子列和，重新开始。这有点像递减的等差数列中，当a(n+1)<0时，则Sn为最大和
